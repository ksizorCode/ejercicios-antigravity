# 📰 NewsReader PWA - Lector de Noticias RSS

Aplicación web progresiva (PWA) diseñada para lectura de noticias en dispositivos móviles con una estética moderna y fluida.

## 🚀 Estructura del Proyecto
- `index.html`: Estructura principal y carga de librerías.
- `assets/css/style.css`: Estilos personalizados y configuración de Tailwind.
- `assets/js/app.js`: Lógica de obtención de datos y renderizado.
- `manifest.json`: Configuración para instalación PWA.
- `sw.js`: Service Worker para soporte offline.

## 🛠️ Tecnologías Utilizadas
- **Tailwind CSS**: Framework de utilidades para diseño rápido y elegante.
- **Font Awesome**: Librería de iconos vectoriales.
- **rss2json API**: Intermediario para convertir feeds XML RSS a formato JSON amigable para JavaScript.
- **Glassmorphism**: Técnica de diseño UI basada en transparencia y desenfoque.

## 📝 Proceso de creación
1. Definición de la interfaz móvil (Mobile First).
2. Implementación de la lógica de conexión con el feed RSS.
3. Aplicación de estilos premium (gradientes, tarjetas de cristal).
4. Configuración del manifiesto y service worker para funcionalidad PWA.
