# Prompt de Usuario 01 🚀

**Fecha:** 21 de Abril, 2026

## Petición original 📝

"pronóstico del tiempo local
Constula una RSS que nos muestre el tiempo en tu zona o permita buscar a través de un input la zona donde te encuentras. Y si no añade un botón que localice mi zona y me de el pronóstico del tiempo"

## Contexto Adicional 💡

El usuario busca una aplicación web moderna que permita:
- Ver el clima local automáticamente (geolocalización).
- Buscar clima de otras zonas mediante input.
- Estética premium con Tailwind CSS y Font-Awesome.
- Explicación didáctica para principiantes.
