# Prompt del Proyecto: PokéDex Pro PWA 👾

Crea una aplicación web progresiva (PWA) que funcione como una PokéDex, obteniendo datos de la API pública PokeAPI.

## Requerimientos:
1. **Tecnología**: HTML5, Vanilla JavaScript y Tailwind CSS.
2. **PWA**: Instalable con `manifest.json` y `sw.js`.
3. **Diseño**: 
   - Estética inspirada en el mundo Pokémon pero con un toque moderno y tecnológico (estilo futurista).
   - Uso de tarjetas de cristal (Glassmorphism).
   - Fondo dinámico que cambie o tenga elementos flotantes (Pokéballs).
   - Iconos de tipos de Pokémon.
4. **Funcionalidad**:
   - Listado de Pokémon con imagen oficial, nombre y número.
   - Barra de búsqueda para filtrar por nombre.
   - Detalle al hacer clic (tipo, habilidades, estadísticas base).
   - Carga infinita o paginación suave.
5. **Comentarios**: Código bien documentado y estructurado.
