# Prompt del Proyecto: Emoji Match Kids PWA 🍦

Crea un juego infantil de memoria donde el objetivo es encontrar parejas de emoticonos idénticos.

## Requerimientos:
1. **Tecnología**: HTML5, Vanilla JS y Tailwind CSS.
2. **PWA**: Instalable con soporte offline.
3. **Diseño**:
   - Estética infantil (colores pastel, bordes muy redondeados).
   - Animaciones divertidas (rebotes, confeti al ganar).
   - Modo noche/día suave.
   - Sonidos simples (opcional, pero con feedback visual).
4. **Funcionalidad**:
   - Tablero de 4x4 o 6x6 (ajustable).
   - Cronómetro y contador de movimientos.
   - Sistema de barajado aleatorio al iniciar.
   - Bloqueo de clics mientras se comprueba una pareja.
5. **Comentarios**: Código muy fácil de seguir para principiantes.
